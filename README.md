# Single-cell RNA-seq Workflow Reproduction

This project reproduces a standard single-cell RNA-seq analysis workflow using Scanpy and the PBMC3k dataset.

## Workflow

- Quality control
- Filtering
- Normalization
- PCA
- Neighborhood graph construction
- UMAP visualization
- Clustering
- Marker gene analysis
- Cell type inference

## Tools

- Python
- Scanpy
- Jupyter Notebook

## Dataset

PBMC3k dataset from 10x Genomics.
